# Full Stack Web Application (React + Vite + Node.js)

This project is a full-stack web application built with **React + Vite** for the frontend and **Node.js (Express)** for the backend. The application provides a fast and scalable architecture for modern web development.

## 🚀 Tech Stack

### Frontend
- React.js
- Vite
- JavaScript (ES6+)
- CSS / Bootstrap

### Backend
- Node.js
- Express.js
- REST API

### Database
- MySQl
